MQTT Shared Subscription Demo

Demonstrates MQTT v5.0 Shared Subscriptions for load balancing across multiple workers.

How Shared Subscriptions Work:
  • Workers subscribe to $share/workers/tasks/+ — messages are distributed (round-robin)
  • Monitor subscribes to tasks/+ (regular) — receives ALL messages
  • Each task is delivered to exactly ONE worker, while the monitor sees everything
