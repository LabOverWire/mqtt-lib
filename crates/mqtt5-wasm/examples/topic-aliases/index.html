MQTT Topic Aliases - Bandwidth Optimizer

Demonstrates MQTT v5.0 Topic Aliases for reducing bandwidth on high-frequency publishing.

How Topic Aliases Work:
  • First publish: Full topic name + alias assignment (e.g., topic="sensors/device-001/temperature/celsius" alias=1)
  • Subsequent publishes: Only alias number (2 bytes instead of full topic string)
  • Long topic names benefit most from aliasing
