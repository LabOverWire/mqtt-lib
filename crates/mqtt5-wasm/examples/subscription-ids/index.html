MQTT Subscription Identifiers Demo

Demonstrates MQTT v5.0 Subscription Identifiers for routing messages based on which subscription matched.

How Subscription IDs Work:
  • Each subscription is assigned a unique identifier (1001, 1002, etc.)
  • When a message arrives, it includes which subscription ID(s) matched
  • A single message can match multiple overlapping subscriptions!
