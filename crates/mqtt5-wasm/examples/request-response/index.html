MQTT Request/Response RPC Demo

Demonstrates MQTT v5.0 Request/Response pattern using ResponseTopic and CorrelationData for RPC-style communication.

How it works:
  • Client sends requests with a unique CorrelationData and ResponseTopic
  • Worker processes requests and replies to the ResponseTopic with matching CorrelationData
  • The tracker shows request→response matching in real-time
