## comnet/

Full research paper evaluating three stream mapping strategies (control-only, per-topic, per-publish) for MQTT over QUIC. Includes the complete experiment infrastructure, raw results, analysis scripts, and publication-ready figures. See [`comnet/README.md`](comnet/README.md) for a comprehensive guide.

## joss/

Short-form software paper describing the mqtt-lib library itself: architecture, features, and usage. Written in Markdown with Pandoc-compatible bibliography (`paper.bib`).
